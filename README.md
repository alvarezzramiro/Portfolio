# Personal Portfolio

This is my personal portfolio website, designed and developed to showcase my projects, skills, and background as a Software Engineering student focused on backend development and scalable systems.

---

## Overview

The portfolio was built with a strong focus on:

- Clean and minimal UI
- Smooth animations and interactions
- Structured and scalable component design
- Clear presentation of projects and skills

It serves as a central place to present my work and track my growth as a developer.

---

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion

---

## Features

- Responsive design (desktop & mobile)
- Animated UI with smooth transitions
- Project showcase with technology badges
- Downloadable CV (English & Spanish)
- Contact section with copy-to-clipboard functionality
- Mobile navigation with interactive menu

---

## Structure

The project follows a modular and scalable structure:

```
src/
 ├── shared/
 │    ├── components/
 │    ├── data/
 │    ├── hooks/
 │    └── utils/
 ├── sections/
 │    ├── home/
 │    ├── about/
 │    ├── skills/
 │    ├── projects/
 │    └── contact/
```
